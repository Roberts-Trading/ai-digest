# AI Digest — 2026-07-18

## Product Updates

### Anthropic — Fable 5 Access Finalized, Meta Compute Deal
- **Summary**: (1) **Fable 5 permanent access structure** announced July 17, effective July 20. After three deadline extensions, Anthropic settled: Max and Team Premium get Fable 5 at 50% of standard weekly limits (which themselves shrink 33% when the Claude Code rate-limit boost ends). Pro and Team Standard lose bundled access — they get a one-time $100 usage credit, then API pricing ($10/$50 per 1M in/out). Anthropic acknowledged demand was "challenging to predict." The reversal from full removal was driven by GPT-5.6 Sol competition and Chinese pricing pressure. (2) **Meta compute deal** — NYT reports Meta in talks to lease AI data center compute to Anthropic, potentially $10B over two years. Anthropic proposed the deal in June. For context: Anthropic is already paying SpaceX $45B over three years for compute ($1.25B/month). The Meta deal is ~1/3 the size of the SpaceX deal.
- **Source**: https://the-decoder.com/anthropic-slashes-claude-fable-5-limits-in-max-and-team-premium-and-pushes-pro-users-toward-api-pricing/ ; https://www.nytimes.com/2026/07/17/technology/meta-anthropic-ai-computing-power.html
- **Significance**: The Fable 5 saga reveals the real constraint: compute capacity, not model capability. Anthropic has the best model but can't serve it to everyone. The $45B SpaceX + potential $10B Meta compute deals show the scale of the capacity race — Anthropic is spending ~$55B over three years just on compute. The Pro tier effectively losing Fable 5 (the $100 credit won't last) creates a two-tier reality: frontier models are now a premium feature, not a baseline.

### OpenAI — "Useful Intelligence per Dollar," GPT-Live, Merged App
- **Summary**: (1) **"A scorecard for the AI age"** — OpenAI's framework: four questions (Is AI completing important work? Cost per successful task? Can results be depended on? Does value per dollar improve?). GPT-5.6 Sol with max reasoning set SOTA on Artificial Analysis Coding Agent Index using 54% fewer output tokens than a leading model. On DeepSWE v1.1, GPT-5.6 Sol hit 72.7% vs Claude Fable 5's 69.9% at 36.2% lower estimated API cost. (2) **GPT-Live-1** — full-duplex voice models (listen and talk simultaneously). Delegates complex tasks to GPT-5.5 while continuing conversation. Available iOS/Android/web. Go/Plus/Pro get GPT-Live-1; free gets mini. No developer API yet. (3) **ChatGPT + Codex merged** into one app with Work mode. 8M+ active users, potentially 9M today. $100 credit promo for $8+/month subscribers. UX complaints: smaller floating window, missing chat history search. (4) **Parental notifications** expanded — parents now notified if teen account deactivated for violence-related violations. Follows Tumbler Ridge shooting incident. Study Mode added to parental controls.
- **Source**: https://openai.com/index/a-scorecard-for-the-ai-age/ ; https://www.deeplearning.ai/the-batch/one-model-talks-another-one-thinks ; https://quasa.io/media/openai-merges-chatgpt-and-codex-into-one-app-after-gpt-5-6-launch-introduces-new-work-mode
- **Significance**: OpenAI's "useful intelligence per dollar" framing is the clearest articulation of the efficiency war. GPT-5.6 Sol beating Fable 5 on DeepSWE at 36% lower cost is a direct competitive shot. GPT-Live's full-duplex architecture (voice model + separate reasoning model) is the technical pattern to watch — it's how the smart speaker will work. The merged app with Work mode is OpenAI's play for the office agent market (vs. Claude Cowork, Cursor Sand).

### SpaceXAI/Cursor — Grok 4.5, grok-build Open-Sourced, Codebase Upload Scandal
- **Summary**: (1) **Grok 4.5** details — MoE ~1.5T params (3× Grok 4.3), 500K context (down from 1M), jointly trained with Cursor on trillions of developer-agent tokens. Pricing: $2/$6 per 1M in/out, $0.50 cached. Trails Fable 5 on most coding benchmarks (SWE-Bench Pro: 64.7% vs 80.4%; DeepSWE 1.1: 53.0% vs 70.0%). Musk claims "roughly comparable to Opus 4.7 but faster." (2) **grok-build open-sourced** — terminal-based coding agent with fullscreen TUI, mouse support, multi-model (GPT-4o, Claude, Grok, Ollama). 17K+ GitHub stars in 24 hours. Apache 2.0. Linux/macOS only (WSL works). (3) **Codebase upload scandal** — Grok Build v0.2.93 secretly transmitted complete Git bundles to xAI-controlled storage. SpaceXAI disabled uploads July 12, open-sourced the harness July 15, committed to deleting data. Default retention now off for all users. (4) **Cursor data pipeline** to Grok training reportedly went live June 28. Standard Privacy Mode routes code to Grok 4.5 training; only legacy strict Privacy Mode retains nothing. Most enterprises on standard.
- **Source**: https://mer.vin/2026/07/grok-4-5-spacexai-and-cursor-frontier-model-benchmarks-and-api-pricing/ ; https://dev.to/jarynagent/spacex-just-open-sourced-a-coding-agent-and-it-hit-17k-stars-in-24-hours-1g2g ; https://winbuzzer.com/2026/07/18/spacexai-open-sources-grok-build-after-upload-backlash-xcxwbn/ ; https://byteiota.com/spacex-buys-cursor-for-60b-what-developers-need-to-know-now/
- **Significance**: The Grok Build upload scandal is the biggest trust breach in coding agents this year. Secretly uploading entire codebases to train a competitor's model, with no disclosure, is exactly the risk the SpaceX-Cursor acquisition raised. The open-sourcing is damage control, not altruism. Grok 4.5's benchmarks confirm it's not competitive with Fable 5 on coding — the Cursor acquisition is about distribution, not model quality. **Action item for any team using Cursor: check Settings > Privacy immediately.** If you're on standard Privacy Mode, your code may already be in Grok training data.

## New Tools & Discoveries

### grok-build (SpaceXAI)
- **What it does**: Terminal-based coding agent with fullscreen TUI, mouse support, multi-model. Apache 2.0. 17K stars in 24 hours.
- **Why notable**: Third category beyond IDE extensions (Cursor) and terminal REPLs (Claude Code, Aider) — terminal-native visual experience. Written in Rust. No Windows support. Default model is Grok (lock-in risk). Plugin ecosystem empty.

### GitHub Copilot SDK
- **What it does**: Copilot's agent engine shipped as an embeddable SDK. Planning, tool calls, file edits, orchestration. Six official bindings: TS, Python, Go, .NET, Rust, Java. GA with semantic versioning. Rust binding landed July 16. ~9,800 stars.
- **Why notable**: GitHub handing out the same engine that powers Copilot CLI. Anyone building a coding agent now starts with GitHub's kernel. BYOK for OpenAI, Azure, Anthropic.

### Microsoft Agent Framework for Go
- **What it does**: Production-grade Go SDK for agentic AI. Merged AutoGen + Semantic Kernel. Tool calling, MCP client, graph-based workflows, human-in-the-loop, checkpointing, OpenTelemetry. Python and .NET hit 1.0 in April; Go preview July 10.
- **Why notable**: Microsoft and Google both betting on Go for agent infrastructure (Google shipped ADK for Go earlier in 2026). Pattern mirrors cloud-native: Kubernetes, Terraform, Docker — all Go. OpenAI and Anthropic still have no Go SDKs.

### Google Cloud Always-On Memory Agent
- **What it does**: 24/7 background agent built with Google ADK + Gemini 3.1 Flash-Lite. No vector DB, no embeddings. LLM reads, thinks, writes structured memory to SQLite. IngestAgent → ConsolidateAgent (every 30 min) → QueryAgent. Cites memory IDs as sources.
- **Why notable**: Replaces RAG with continuous LLM consolidation. The "sleep cycle" consolidation pattern (reviewing unconsolidated memories every 30 min, finding connections) is a novel architecture. Whether it scales beyond a reference implementation is the question.

### Ledger Agent Stack
- **What it does**: Open-source framework for AI agents to manage crypto without accessing private keys. Every transaction requires physical approval on Ledger hardware wallet. Also secures AI credentials and acts as auth key for GitHub, Discord, 1Password.
- **Why notable**: Hardware-enforced approval for AI agent transactions. The "agents propose, humans approve" pattern is the right security model for autonomous financial agents. Mitigates prompt injection by requiring physical confirmation.

- **Fable 5 access**: If you're on Pro or Team Standard, the $100 credit is a stopgap. Plan migration to Sonnet 5 for routine work, reserve Fable 5 for tasks that need it. Max/Team Premium users: audit your weekly usage against the 50% cap.

- **GPT-5.6 Sol**: If you're optimizing for coding cost, GPT-5.6 Sol beats Fable 5 on DeepSWE at 36% lower cost. Worth routing long-horizon engineering tasks here.

- **GitHub Copilot SDK**: If you're building a coding agent, start with GitHub's kernel instead of reinventing orchestration. Six language bindings, GA, BYOK.
