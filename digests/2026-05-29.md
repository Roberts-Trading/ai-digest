# AI Digest — 2026-05-29

## Product Updates

### Anthropic — Valuation Now $965B, Opus 4.8 Coverage Continues
- **Summary**: The Anthropic $65B raise and Opus 4.8 launch dominated day-two coverage. Fortune, Al Jazeera, and AI News all confirm Anthropic has leapfrogged OpenAI ($852B) as the most valuable private AI company. Claude getting 1M+ new signups/day. The Trump administration has labeled Anthropic a "supply chain risk" over its refusal to allow unrestricted military access. Opus 4.8 framed as "modest but tangible" improvement — effort control, dynamic workflows, and Messages API system entries all getting detailed coverage.
- **Source**: https://fortune.com/2026/05/29/anthropic-raises-65-billion-at-record-965-billion-valuation-promises-mythos-ai-model-in-wide-release-in-coming-weeks-releases-claude-opus-4-8/ ; https://www.aljazeera.com/economy/2026/5/29/anthropic-soars-to-965bn-valuation-leapfrogging-openai
- **Significance**: The "supply chain risk" label from the US government is a new escalation in the Anthropic-Pentagon fight. It's one thing for the DoD to sign deals with other companies. It's another for the administration to formally designate a company as a risk. This could affect government contracts and procurement beyond just military AI.

### OpenAI — Codex on Windows, Canvas Sunset, CPA Ads
- **Summary**: Three moves. (1) Codex computer use now available on Windows — can see your screen and perform tasks on your device. Remote job management via the ChatGPT app. (2) Canvas interface being sunset — no longer available with GPT-5.5 Instant or Thinking. Legacy model access for a "limited time." (3) CPA ads confirmed live — advertisers pay only when users take a specific action. Early access to conversion-optimized campaigns by June 5.
- **Source**: https://www.theverge.com/ai-artificial-intelligence/940095/openais-codex-can-now-control-your-windows-computer-too ; https://www.theverge.com/ai-artificial-intelligence/940028/openai-is-sunsetting-chatgpts-canvas-interface
- **Significance**: Canvas being sunset is a signal — OpenAI is consolidating interaction models. The side-by-side editing paradigm lost to the agentic coding paradigm (Claude Code, Codex CLI). Codex on Windows + remote management is the "agent that works while you're away" pattern.

### Cursor — Auto-Review Mode
- **Summary**: Cursor shipped Auto-Review Mode, reducing agent approval prompts during coding sessions. Agents execute tool calls without asking for permission on every step, while keeping safety guardrails on execution. Interactive web demo lets you test the feature before applying it to your codebase.
- **Source**: https://vibin.live/cursor-ships-auto-review-mode-to-cut-agent-approval-interruptions
- **Significance**: Every coding agent is solving the same problem: how to balance autonomy with safety. Cursor's answer is less prompting, more background checking. Same direction as Claude Code's dynamic workflows — trust the agent, verify the output.

### xAI — Grok Build API Launch
- **Summary**: grok-build-0.1 now available on the xAI API in public beta. 256K context window, $1/$2 per 1M in/out tokens, $0.20 cached input. Over 100 tokens/second. Available to all SuperGrok and X Premium+ subscribers.
- **Source**: https://toolnav.io/news/2026-05-30-xai-grok-build-coding-agent-cli/
- **Significance**: The pricing is aggressive — Opus 4.8 costs $5/$25, Grok Build costs $1/$2. Five times cheaper per token. If quality is "good enough" for routine coding tasks, this undercuts the entire frontier model pricing structure.

### Microsoft — Power Pages Plugin GA for Claude Code + Copilot
- **Summary**: Power Pages plugin for GitHub Copilot CLI and Claude Code went GA. Covers full site lifecycle: creation, deployment, data modeling, web API, server logic, ALM, and security hardening — all through conversational workflow. New commands: `/create-site`, `/security-review`, `/plan-alm`.
- **Source**: https://www.microsoft.com/en-us/power-platform/blog/power-pages/build-power-pages-sites-with-ai-through-agentic-coding-tools-now-generally-available/
- **Significance**: Microsoft shipping Claude Code plugins alongside Copilot is notable. It's not "Claude OR Copilot" — it's "both, depending on the task." The multi-agent ecosystem is becoming real.

## People & Commentary

### Derek Thompson — "The Great AI Cost Panic of 2026"
- **What they said/did**: One client spent $500M on Claude in a single month. Average business spending 13x more on AI tokens than January 2025. Uber and Microsoft reportedly dropped some Claude Code licenses as costs became unmanageable. JPMorgan published a note: "AI Token Costs are Eating Internet Profits Alive." Shopify, Spotify, ServiceNow, Roku all report AI surging as share of operating expenditures.
- **Source**: https://www.derekthompson.org/p/the-great-ai-cost-panic-of-2026
- **Context**: This is the most detailed cost-panic piece yet. The $500M/month figure is staggering. The 13x spending increase in 18 months is unsustainable if ROI doesn't materialize. Thompson frames this as a potential bubble pop — but also notes the parallel to early cloud computing, where costs felt insane before they normalized.

### CNBC — "Tokens or Humans: The New Corporate Trade-Off"
- **What they said/did**: CFOs trading future headcount for tokens. 95% of enterprise AI still runs on the priciest frontier models even for simple tasks. Three phases: boards demand AI action → tokenmaxxing → reassessing premium model needs. 10x savings possible with proper model routing.
- **Source**: https://www.cnbc.com/2026/05/29/-tokens-or-humans-the-new-corporate-trade-off.html
- **Context**: The 95% figure is the actionable number. Most enterprises are burning Opus-class tokens on tasks that could run on Haiku or Sonnet. Model routing is the lowest-hanging fruit for cost optimization.

### Josh Bersin — AI Prices Going Up, Up, Up
- **What they said/did**: AI tool prices will skyrocket. "Giving everyone Claude" may stop, just like "giving everyone a PC" didn't happen fast. AI will be treated as investment, not random tool. The shift from assistants to agents to superagents accelerates when agents cost $50K/month to run.
- **Source**: https://joshbersin.com/2026/05/ai-prices-are-going-up-up-up-and-what-this-means-for-enterprise-ai/
- **Context**: Bersin's framing is different from the cost-panic crowd. He argues prices will go UP because value is being demonstrated, not down. The implication: premium models get more expensive, budget models get commoditized, and the middle disappears.
