# AI Digest — 2026-05-24

## Product Updates

### Letta (MemGPT)
- **Summary**: Letta continues to be recognized as the leading self-improving agent runtime, scoring highest on LongMemEval (83.2%) among memory tools. New comparison articles position it against Mem0, Zep, and Cognee.
- **Source**: https://niteagent.com/blog/ai-agent-memory-comparison-2026/
- **Significance**: Letta's tiered memory architecture is becoming the reference design for stateful agents. The self-improving capability is now a recognized differentiator.

### Anthropic Claude
- **Summary**: Two major launches at "Code with Claude" conference in London (May 19): self-hosted sandboxes (public beta) and MCP tunnels (research preview). Also launched Claude Mythos Preview, which found 10,000+ critical vulnerabilities in system software.
- **Source**: https://business-news-today.com/anthropic-moves-claude-agents-inside-the-customer-perimeter-with-self-hosted-sandboxes-and-mcp-tunnels/
- **Significance**: Self-hosted sandboxes solve the enterprise compliance problem — agents can now run inside your security perimeter. MCP tunnels kill the need for inbound firewall exceptions. Mythos Preview dramatically changes the security landscape.

### Claude Code
- **Summary**: Version 2.1.147 adds a Workflow tool (behind feature flag) for deterministic multi-agent execution sequences inside the CLI. Weekly release cadence continues.
- **Source**: https://www.joinnextdev.com/blog/claude-code-21147-turns-agent-workflows-into-infrastructure
- **Significance**: Anthropic is positioning Claude Code as programmable infrastructure, not just a smarter autocomplete. The Workflow tool compresses the need for separate orchestration frameworks.

### Cursor
- **Summary**: Composer 2.5 launched as new default model. Trained on 25x more synthetic RL tasks, matches Opus 4.7 and GPT-5.5 on benchmarks at 1/10th the price. Also launched Python SDK for programmatic agent control.
- **Source**: https://forum.cursor.com/t/composer-2-5-is-now-live/160934
- **Significance**: Cursor is now training a model from scratch with SpaceXAI on Colossus 2 (10x more compute). The IDE-to-model-lab transition is accelerating. The Python SDK makes agentic coding a headless, scriptable tool.

### OpenAI ChatGPT
- **Summary**: PowerPoint integration (beta), SynthID + C2PA watermarks on all generated images, ChatGPT Pro personal finance tools via Plaid, flight/package tracking, and a new Library feature for file management.
- **Source**: https://cryptobriefing.com/chatgpt-powerpoint-integration-presentation-editing/
- **Significance**: OpenAI is pushing ChatGPT into full enterprise productivity. The invisible watermarks (SynthID + C2PA) are a compliance play ahead of EU AI Act and California AI Transparency Act enforcement in August.

## New Tools & Discoveries

### Docker Gordon
- **What it does**: AI agent for the entire container workflow — built into Docker Desktop 4.74+ and CLI. Sees your environment, proposes fixes, takes action with approval.
- **Source**: https://www.docker.com/blog/meet-gordon-dockers-ai-agent-for-your-entire-container-workflow/
- **Why notable**: First major AI agent integrated natively into Docker. Free to start, reduces problem-to-fix time dramatically.

### Gemini Managed Agents
- **What it does**: Google's Gemini API now supports managed agents — spin up an agent in a secure cloud sandbox with one API call. Uses AGENTS.md and SKILL.md for agent definition.
- **Source**: https://blog.google/innovation-and-ai/technology/developers-tools/managed-agents-gemini-api/
- **Why notable**: The AGENTS.md/SKILL.md pattern is converging across platforms. Google is making agent infrastructure a first-class API primitive.

### Grok Build
- **What it does**: xAI's new AI coding agent, focused on developers and crypto projects. Supports parallel agents for simultaneous tasks.
- **Source**: https://www.analyticsinsight.net/news/what-grok-build-means-for-the-future-of-ai-coding-tools
- **Why notable**: Three-way race now: Claude Code vs Codex CLI vs Grok Build. xAI is late but bringing the Cursor/SpaceX ecosystem with it.

### Kore.ai Artemis
- **What it does**: Enterprise agent platform with Agent Blueprint Language (ABL) — a YAML-based declarative language for defining agents. The "Arch" AI system translates business requirements into production-ready ABL.
- **Source**: https://venturebeat.com/technology/kore-ai-launches-artemis-ai-agent-platform-expands-challenge-to-microsoft-and-salesforce
- **Why notable**: ABL is an interesting intermediate representation between natural language and production infrastructure. Could define how agents are serialized and shared.

### AWS MCP Server (GA)
- **What it does**: Now generally available with full API coverage. IAM-based access controls, CloudTrail logging, sandboxed Python execution. Works with Claude Code, Cursor, Codex.
- **Source**: https://www.infoq.com/news/2026/05/aws-mcp-ga/
- **Why notable**: MCP is becoming the standard way agents connect to cloud services. AWS going GA means enterprise-grade agent-to-AWS access is real.
