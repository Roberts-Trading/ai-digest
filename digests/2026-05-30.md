## New Tools & Discoveries

### Skills as the New Packages
- **What it does**: DEV Community article argues that the SKILL.md format (originally developed by Anthropic, now adopted by Claude Code, Cursor, Codex, Gemini CLI, GitHub Copilot, Goose, OpenHands, Letta, and dozens more) is becoming the industry's first portable agent instruction standard. Hermes agent demonstrates: after completing a task, it crystallizes the approach into a new skill.
- **Source**: https://dev.to/rajdhokai/skills-are-the-new-packages-2e81
- **Why notable**: This directly validates the approach Letta Code uses. Skills are the portable asset — the agent is ephemeral. The fact that competitors who disagree on everything agreed on a folder with a SKILL.md in it is the most important convergence in agent tooling.

### Agent Memory: Supersede, Don't Delete
- **What it does**: New pattern emerging: stale facts should be marked `superseded` (with reason and successor reference) rather than deleted. Preserves history and lets agents answer "what changed?" instead of just "what is?"
- **Source**: https://dev.to/tokidigital/three-agent-memory-threads-this-week-one-missing-field-310e
- **Why notable**: This is a memory architecture improvement Letta should consider. Currently Letta's `memory_replace` is a plain string swap — no change history. The `superseded` pattern adds temporal reasoning capability to memory.
