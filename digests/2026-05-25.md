# AI Digest — 2026-05-25

## Product Updates

### Letta (MemGPT)
- **Summary**: New comprehensive analysis of AI agent memory architectures positions Letta's three-tier model (core/archival/recall) as the reference design. Anthropic shipped "Dreaming" — a consolidation process for Managed Agents — following Letta's principle that memory management is an explicit agent action, not passive injection.
- **Source**: https://www.digitalapplied.com/blog/ai-agent-memory-vector-graph-episodic-2026
- **Significance**: The "Dreaming" primitive validates Letta's architecture: consolidation is scheduled, not passive. The memory-first agent pattern is becoming the industry standard.

### Anthropic Claude
- **Summary**: Major leaks — Claude Mythos 1 and Opus 4.8 spotted in backend systems. "Conway" appears to be a persistent background AI agent for long-running tasks. Also: official Claude Code plugin directory launched on GitHub (20k+ stars in 3 days), with "Anthropic Verified" badge for vetted plugins.
- **Source**: https://thewincentral.com/claude-mythos-1-and-opus-4-8-leak-online-anthropic-may-be-preparing-its-biggest-ai-upgrade-yet/
- **Significance**: Anthropic is preparing a full ecosystem launch — not just a model. Conway (persistent agent) + Mythos (security) + Opus 4.8 (frontier reasoning) + plugin directory = Anthropic building an operating layer, not just a chat API.

### Project Glasswing Update
- **Summary**: Anthropic published an update on Project Glasswing. Claude Mythos Preview identified thousands of vulnerabilities including in wolfSSL (patched). The defensive coalition now includes AWS, Apple, Google, Microsoft, CrowdStrike, and NVIDIA. Mythos is not publicly released — access restricted to the coalition.
- **Source**: https://dig.watch/updates/anthropic-says-ai-system-identified-thousands-of-critical-software-flaws
- **Significance**: The coalition model is novel — restricting access to defensive partners only. This is Anthropic's answer to the dual-use problem: powerful vulnerability discovery without arming attackers.

### Cursor
- **Summary**: New detailed review and pricing analysis confirms Cursor Composer 2.5 is best-in-class for tab completion and small refactors, but falls behind Claude Code on large 800+ line multi-file changes. Pro plan ($20/mo) is credit-pool based, not unlimited — most heavy users end up on Pro+ ($60/mo).
- **Source**: https://dev.to/rikuq/cursor-review-2026-honest-not-for-me-take-from-a-vscode-user-931
- **Significance**: The Cursor vs Claude Code split is becoming a real architectural decision: if you hand-code with AI assist, use Cursor. If you direct AI to ship features, use Claude Code. The workflows are diverging.

### OpenAI ChatGPT
- **Summary**: ChatGPT Ads Manager gets daily budgets, granular US geo-targeting (state/DMA/ZIP), and aggregate reporting. PowerPoint integration (beta) launched May 24. Microsoft 365 prices rising July 1 (Business Basic +16%, Business Standard +12%, new E7 tier at €92/user/mo).
- **Source**: https://www.thekeyword.co/news/openai-chatgpt-ads-manager-daily-budgets
- **Significance**: ChatGPT is becoming a full ad platform competing with Google/Meta. The Microsoft 365 price hikes signal that AI features are being monetized through seat pricing, not just API costs.

## New Tools & Discoveries

### Thunderbit Distill + Extract
- **What it does**: High-fidelity HTML-to-Markdown engine (0.87 ROUGE-L) plus structured JSON/CSV extraction from URLs. Ships with MCP server and CLI. 100k+ users already.
- **Source**: https://thunderbit.com/docs
- **Why notable**: MCP server means it plugs directly into Claude Code, Cursor, and other agent tools. Clean Markdown from complex web pages is exactly what AI agents need for RAG pipelines.

### MLflow AI Gateway for Claude Code
- **What it does**: Route Claude Code through MLflow 3.12.0 gateway for automatic request tracing, budget enforcement, and guardrails (prompt injection screening, PII redaction). Two environment variables, zero code changes.
- **Source**: https://mlflow.org/blog/gateway-claude-code/
- **Why notable**: Solves the governance problem for autonomous coding agents. If you're running Claude Code in production, you need observability and budget limits — this is the easiest way to add both.

### Fujitsu Self-Evolving Multi-AI Agent
- **What it does**: Multiple AI agents that learn from daily execution results, human feedback, and policy changes. Agents identify reasons for success/failure and automatically adjust prompts and evaluation criteria.
- **Source**: https://global.fujitsu/en-global/pr/news/2026/05/25-01
- **Why notable**: Enterprise take on the same pattern Letta uses — agents that actively manage their own behavior. The "self-evolving" framing is different from "self-improving" but the core idea is the same: agents that get better at their job without human prompt engineering.
