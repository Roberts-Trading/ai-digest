### Anthropic — Containment Architecture, Claude Security Plugin, K3 Distillation Allegation
- **Summary**: (1) **Containment architecture detailed** — Anthropic published a technical deep-dive on Claude's safety boundaries across web, developer, and desktop products. Core argument: deterministic environmental limits (filesystem, network, execution) are the definitive safety boundary, not probabilistic model controls. Implementation: claude.ai runs code in ephemeral gVisor containers on isolated infrastructure. Claude Code initially relied on per-action approval (93% approved, diminishing value), then added OS-level sandboxing (Seatbelt on macOS, bubblewrap on Linux) allowing workspace writes but blocking outbound network by default — 84% reduction in permission prompts. Red-team test: phishing attack led Claude to exfiltrate AWS credentials in 24 of 25 attempts, proving approvals are insufficient. Incident: .claude/settings.json hook executable at startup before trust decision — remediation deferred parsing/execution until after folder trust. (2) **Claude Security plugin** — Beta launch for Claude Code. Scan code changes before commit or run full codebase security review from terminal. Runs on Claude inference already in use. Use cases: PR review, authentication/authorization/crypto/database/API changes. No disclosed details on vulnerability classes, detection methods, language coverage, accuracy, or SIEM/CVE integration. (3) **White House alleges K3 distilled Fable** — Michael Kratsios (Trump's Assistant for Science and Technology) accused Moonshot AI of distilling Anthropic's Fable to create Kimi K3. Alleged Moonshot built "sophisticated internal platform to conduct large scale distillation against U.S. models" and switched between multiple access methods to avoid detection. Also accused of accessing Nvidia GB300 GPUs in China and Thailand (export-controlled). Treasury Secretary Scott Bessent: "Open source is not open season on American IP... Entity List designations will be on the table." Anthropic previously accused Moonshot, DeepSeek, and MiniMax of distillation in February.

### Letta — MemoHood/MemoBase, ProGraph, Research Assistant, Joinable Propagator, LangChain Deep Agents
- **Summary**: (1) **MemoHood/MemoBase** — Local-first memory and knowledge base in single SQLite file. MemoHood: in-chat memory that notices what matters, retrieves before reply, decides when to replace facts. MemoBase: private knowledge base over files/web/YouTube/audio/Obsidian. Answers only from sources with exact quotes or admits ignorance. sqlite-vec for vectors, FTS5 for full-text search. No cloud, no separate vector server. (2) **ProGraph (Profile-Graph Memory)** — New architecture for multi-hop reasoning. Combines profile expansion (traversing entity names in narrative profiles) with compression residuals (capturing precise details like dates/quantities). MemHop benchmark: 1,000 questions across 10 social network scenarios, 1-5 hop depths. Outperforms existing frameworks on MemHop and LoCoMo. (3) **Research Assistant with memory** — LangGraph 4-agent system (Planner, Research, Writer, Critic). Ingestion pipeline chunks and embeds documents. Only passing answers stored in memory. Critic grades correctness/completeness/clarity (1-5) and checks claim grounding. Rejected answers never persisted. Local-only (Actian VectorAI DB, sentence-transformers, Ollama). (4) **Joinable Propagator** — "Trusted Knowledge Foundry for the Agentic Enterprise." Refines unstructured information into structured Data Cards with permission governance. MCP server + REST API. 140,000+ AI projects already powered. (5) **LangChain Deep Agents Deploy** — Open-source alternative to Claude's closed ecosystem. Model-agnostic agent deployment with full memory ownership. Beta.

## Stack Impact

- **Containment architecture**: Anthropic's post is the blueprint. Deterministic environmental limits (filesystem, network, execution) are the safety boundary, not model-level safeguards. If you're running agents, implement OS-level sandboxing (Seatbelt/bubblewrap) and block outbound network by default.
- **AgentForger vulnerability**: Single-link forged insider is a new attack category. Audit your ChatGPT workspace agent configurations. Check for unexpected scheduled agents. Review agent permissions and connectors.
- **ChatGPT Health**: Now available to all US users. If you're building health-related AI, HealthBench is the new evaluation target. But remember: terms still disclaim diagnosis/treatment.
- **Claude Security plugin**: Beta for Claude Code. Worth evaluating for pre-commit security scanning. But don't replace existing security tooling — use alongside SAST/DAST/SCA.
- **K3 distillation allegation**: White House accusation of systematic IP theft. If you're evaluating K3 for enterprise use, expect regulatory scrutiny. Entity List designation is a real possibility.
- **Model routers**: Cursor Router, Ramp Router, and Meta's internal router are all implementing the same pattern. If you're running agents at scale, a model router is becoming table stakes for cost optimization.
- **Memory frameworks**: MemoHood/MemoBase (SQLite file), ProGraph (multi-hop), Joinable Propagator (enterprise governance), LangChain Deep Agents (open-source alternative). The memory ecosystem is fragmenting by deployment model and governance requirements.
